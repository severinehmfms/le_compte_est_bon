
Titre : Le compte est bon

Auteur : Séverine Hori Maitrehut

Description : 
Ce programme simule le jeu le compte est bon. 
L'ordinateur définit un nombre aléatoire entre 101 et 999.
Le joueur doit essayer d'obtenir ce nombre grâce à plusieurs opérations que le programme lui propose d'effectuer.
Pour cela, l'ordinateur propose au joueur 6 plaques aléatoires, chacune contenant un nombre ; et la possibilité de choisir une opération : addition, soustraction, multiplication ou division.
Le joueur doit essayer d'obtenir le nombre exact ou de s'en rapprocher au maximum.
